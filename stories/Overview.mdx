# Payload CMS UI Component Library

Welcome to the Payload CMS UI component library! This Storybook showcases all the reusable UI components from the Payload CMS admin interface.

## Component Organization

### UI Components (`@payloadcms/ui`)

#### Elements
Core UI elements that form the building blocks of the admin interface:
- **Button** - Primary button component with various styles and sizes
- **Modal** - Overlay component for dialogs and confirmations
- **Table** - Data table component for displaying collections
- **Loading** - Loading states and spinners
- **Tooltip** - Contextual help tooltips

#### Fields
Form field components used in document editing:
- **Text** - Text input field
- **Textarea** - Multi-line text input
- **Number** - Numeric input field
- **Email** - Email address input
- **Password** - Password input with visibility toggle
- **Checkbox** - Checkbox input
- **RadioGroup** - Radio button group
- **Select** - Dropdown selection field
- **DateTime** - Date and time picker
- **Upload** - File upload field
- **Relationship** - Relationship field for connecting documents

#### Icons
SVG icon components used throughout the interface:
- **Plus** - Add/create icon
- **Edit** - Edit icon
- **Trash** - Delete icon
- **Search** - Search icon
- **Chevron** - Navigation arrows
- **And many more...**

#### Providers
React context providers for application state:
- **ThemeProvider** - Light/dark theme management
- **AuthProvider** - User authentication state
- **ConfigProvider** - Payload configuration context
- **FormProvider** - Form state management

### Next.js Components (`@payloadcms/next`)

#### Elements
Next.js specific components:
- **Nav** - Main navigation component
- **DocumentHeader** - Document editing header
- **FormHeader** - Form header with breadcrumbs

#### Layouts
Page layout components:
- **Default** - Standard admin layout
- **Minimal** - Minimal layout for auth pages

## Design System

### Colors
Payload uses a carefully crafted color system with support for both light and dark themes.

### Typography
The interface uses the Inter font family for optimal readability.

### Spacing
Consistent spacing system based on 4px grid.

## Getting Started
